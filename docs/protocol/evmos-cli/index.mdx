# Create a Launchpad

Here's how to Launch a Presale on GasPad: 

**Step 1: Verify Token**

1. Connect your wallet.
2. Go to the GasPad website.
3. Input your token address, if you already have one, or create a new token for the Launchpad.
4. Select BNB (if you want to raise funds using a stable coin for your presale).
5. Select a fee choice.
6. After selecting, the token address will be shown and you can click on “Approve”.
7. MetaMask will now ask you to confirm the transaction and show you the fee that you are required to pay
